    Starts this SealedPiping and returns either the result of type A if A is a scala.concurrent.Future, or otherwise Try[A].

    Starts this SealedPiping and returns either the result of type A if A is a scala.concurrent.Future, or otherwise Try[A].

    If the stream runs synchronously the call will not return before the stream has finished running completely. In this case any returned scala.concurrent.Future will be already completed. Otherwise, if the stream runs asynchronously, it will return (more or less) immediately and the stream will run detached from the caller thread.
