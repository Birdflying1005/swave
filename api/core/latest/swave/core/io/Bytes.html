Bytes 

Companion object Bytes

trait Bytes[T] extends AnyRef

Typeclass qualifying T as a type holding one or more bytes.

Used as an abstraction across popular immutable "byte sequence types" like akka.util.ByteString or scodec.bits.ByteVector.

The API surface is modelled mostly after scodec.bits.ByteVector.
