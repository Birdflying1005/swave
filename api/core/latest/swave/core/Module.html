Module

Related Docs: object Module | package core

abstract class Module[I <: Input, O <: Output] extends AnyRef

A Streaming Component with up to four interfaces:

  • A Top input
  • A Top output
  • A Bottom input
  • A Bottom output

IT OT ┌──▼──┬──▲──┐ │ │ │ │ │ │ └──▼──┴──▲──┘ OB IB

A Module without output interfaces might produce a result of type R.
